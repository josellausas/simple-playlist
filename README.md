A Simple Playlist Mobile App

### Dependencies
- React Native
- XCode
- Android Studio
- Cocoapods

# Install

`npm install` or `yarn install`
then `cd ios` and `pod install`

## Run iOS
`npx react-native run-ios`

## Run Android
`npx react-native run-android`
